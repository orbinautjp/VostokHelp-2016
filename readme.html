このファイルについて

Vostok MODの日本語操作ガイドです。
Orbiter 2016で使用できます。

ゲーム内で日本語の操作ガイドを利用できるようになります。
また、Launchpadに表示されるシナリオの説明文も日本語化されます。

インストール

HtmlとScenariosの2つのフォルダをコピーする。
Orbiterのインストールフォルダにペーストして上書き。

使用方法

Orbiterを起動する。
LaunchpadでVostok_JPフォルダをダブルクリック。
使用したいシナリオをダブルクリックして、ゲームを開始する。

ゲーム内で、Alt+F1を同時押しする。

注意

この操作ガイドの作者と、Vostok MODの製作者は無関係です。
本ファイルを使用したことによる不具合について、MOD製作者に質問しないようお願いします。

シナリオ説明文の読み込みに少し時間がかかりますが、Orbiter2016の仕様です。
